== Installation ==


= Modern Way: =
1. Go to the WordPress Dashboard "Add New Plugin" section.
2. Search For "Block Referral Spam". 
3. Install, then Activate it.

= Old Way: =
1. Upload the `block-referral-spam` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress.


= How To Enable Block Referral Spam: =

Activate the Plugin. You are protected, no setting or configiration.

You are done!
